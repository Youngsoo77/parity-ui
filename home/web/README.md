# Parity Web UI - Top Bar and Home Page

## Development - topbar

```bash
$ npm i
$ npm start
$ open http://localhost:3010 || xdg-open http://localhost:3010
```

## Development - home page
```bash
$ npm i
$ npm start
$ open http://localhost:3010/home.html || xdg-open http://localhost:3010/home.html
```
